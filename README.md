# YOLO CODE 模型训练平台

基于YOLO模型的一体化深度学习平台，覆盖**数据集管理 → 手动标注 → 模型训练 → 模型推理 → 模型评估 → 模型导出**的完整工作流。

---

## 功能模块

| 模块 | 图标 | 功能描述 |
|------|------|---------|
| 仪表盘 | 📊 | 项目概览、工作目录管理、系统状态、快捷操作 |
| 手动标注 | 📝 | 手动绘制YOLO格式边界框，类别管理，标注保存 |
| 数据集 | 📁 | 创建/导入数据集，训练验证测试集分割，多格式导出 |
| 模型训练 | 🚀 | 环境检测修复、YOLO训练、实时曲线、指标监控 |
| 模型推理 | 🔍 | 图片/视频/摄像头检测，结果导出CSV/JSON |
| 模型评估 | 📈 | mAP/Precision/Recall评估，模型格式导出 |
| 模型导出 | 📤 | ONNX/TensorRT/OpenVINO/TFLite/TorchScript/CoreML |
| 关于 | ℹ️ | 版本信息 |
| 终端 | 💻 | Linux风格命令行，支持pip/Python/系统命令 |

## 正常使用流程

以下为从零开始完成一个目标检测项目的完整步骤：

### 第一步：设置工作目录

启动软件后进入**仪表盘**，点击蓝色 🏠 按钮设置工作目录。所有数据集、模型、标注、推理结果都将保存在此目录下。

### 第二步：准备数据集

1. 切换到**数据集**模块
2. 输入数据集名称，点击"创建数据集"
3. 点击"导入数据"选择图片文件夹，或直接将图片放入工作目录的 `datasets/<名称>/images/` 下
4. 如需分割训练/验证集，设置比例后点击"执行分割"

### 第三步：标注数据

1. 切换到**手动标注**模块
2. 点击"浏览"选择数据集中的图片目录
3. 添加类别名称（如 dog, cat, person）
4. 在图片上拖拽鼠标绘制标注框
5. 按 Delete 删除选中框，鼠标滚轮缩放
6. 点击"保存所有标注"，标注文件保存为YOLO格式到 `labels/` 目录

### 第四步：训练模型

1. 切换到**模型训练**模块
2. 点击"检测环境"确认系统状态
3. 选择数据集的 `data.yaml` 文件
4. 选择模型（推荐 yolov8n 入门）
5. 调整参数后点击"开始训练"
6. 在**训练监控**页查看实时曲线和指标
7. 训练完成后模型保存在 `runs/train/` 下

### 第五步：推理测试

1. 切换到**模型推理**模块
2. 浏览选择训练好的 `.pt` 模型并加载
3. 选择输入源（图片/视频/摄像头）
4. 调整置信度阈值
5. 点击"开始推理"查看检测结果
6. 可导出结果为CSV或JSON

### 第六步：评估与部署

1. 在**模型评估**模块评估模型性能
2. 在**模型导出**模块将模型转换为部署格式（ONNX推荐）

## 系统要求

- **操作系统**: Windows 10+ / Linux / macOS
- **Python版本**: 3.10 或更高
- **推荐硬件**: NVIDIA GPU (CUDA支持) 或 CPU
- **内存**: 8GB 以上

---

## 安装指南

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 安装GPU版PyTorch（可选）

有NVIDIA GPU时推荐，在软件内"模型训练 → 安装PyTorch"也可完成。

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

### 3. 启动

**三端通用** — Windows / Linux / macOS 统一入口：

```bash
python start.py
```

启动器自动检测 Python 版本、检查依赖并安装缺失项，然后启动软件。双击 `start.py` 也可运行（需关联 `.py` 到 Python）。

## 常见问题

### PyTorch加载失败(c10.dll)
软件内操作：**模型训练 → 环境操作 → 修复PyTorch**

### GPU不显示
- 安装NVIDIA驱动
- **模型训练 → 安装PyTorch** 选择CUDA版本

### 训练显存不足
- 减小批次大小（如8或4）
- 减小图片尺寸（如320）
- 使用更小模型（yolov8n）

### 标注颜色异常
图片颜色通道已校准，如仍有问题请检查图片格式。

### 下载速度慢
**模型训练 → 镜像源配置** 切换清华或阿里云镜像。
